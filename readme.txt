A widget to display recent posts by category.

== Description ==

This is an improvement of the default Recent Posts widget to select posts by a category.

For more informations and news please visit the official [project page](http://mgiulio.info/projects/mg-recent-posts/).

== Installation ==

The standard procedures for WP plugins installation apply:

1. Upload the plugin to your WordPress site
1. Activate the plugin through the 'Plugins' menu in WordPress
